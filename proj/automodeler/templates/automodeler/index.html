Welcome to Modelworx!

Modelworx is a tool developed as part of a project for Penn State University. The intent of Modelworx is to provide automation for preprocessing datasets, and automatically building models from the scikit-learn Python package. Modelworx as a tool will attempt to determine the best model to be used for your dataset! Additionally, Modelworx will create an URI endpoint that can be programatically utilized to run models remotely.

To get started, create an account, login, and then upload your first dataset!
